# C++ Labs - Podstawy programowania w C/C++

Repozytorium z rozwiazaniami zadan laboratoryjnych.
Kazda labka ma wlasny katalog `Kx_...` z plikami `.cpp` i wlasnym `CMakeLists.txt`.

## Struktura

```
cpp-labs/
├── K5_OOP_Cykl_Zycia_Dziedziczenie/   # cykl zycia, dziedziczenie, polimorfizm
│   ├── zad1_...cpp ... zad9_...cpp
│   ├── CMakeLists.txt
│   └── README.md
```
