# C++ Labs - Podstawy programowania w C/C++

Repozytorium z rozwiazaniami zadan laboratoryjnych.
Kazda labka ma wlasny katalog `Kx_...` z plikami `.cpp` i wlasnym `CMakeLists.txt`.

## Struktura

```
cpp-labs/
├── K5_OOP_Cykl_Zycia_Dziedziczenie/   # cykl zycia, dziedziczenie, polimorfizm
│   ├── zad1_...cpp ... zad9_...cpp
│   ├── CMakeLists.txt
│   └── README.md
└── (kolejne labki: K6, K7, ...)
```

## Kolejne zajecia

- **K6**: Przeciazanie operatorow, Szablony, Wyjatki.

## Jak dodawac nowa labke

1. Stworz katalog `Kx_Temat/`.
2. Wrzuc pliki `.cpp` (jeden plik = jedno zadanie).
3. Skopiuj `CMakeLists.txt` z poprzedniej labki i podmien nazwy plikow.
4. Dopisz wpis do tej tabeli.
